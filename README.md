Quick description

### Documentation and code

URLs for the docs and code.

### Installation

For `conda` users you can

```shell
conda install --channel conda-forge xenso
```

or, if you are a `pip` users

```shell
pip install xenso
```

### Example

```python
import xenso


xenso.meaning_of_life_url()
```
